# 문제 #
당신은 음식점의 계산을 도와주는 점원이다. 카운터에는 거스름돈으로 사용할 500원, 100원, 50원, 10원짜리 동전히 무한히 존재한다고 가정한다. 소닌ㅁ에게 거슬러 줘야 할 돈이 N원이리 때 거슬러줘야 할 동전의 최소 개수를 구하라. 단, 거슬러 줘야 할 돈은 항상 10의 배수이다.

문제 팁: '가장 큰 화폐 단위부터' 돈을 거슬러 주기
